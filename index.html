ホームページ更新マニュアル

河野研究室 公式サイト

このWebサイトは、デザインや複雑なプログラムを触ることなく、文字データ（data.js）を書き換えるだけで自動的に更新されるように構築されています。
以下の手順に沿って、テキストエディタ（メモ帳など）で編集を行ってください。

目次

  • 0. 編集時の「絶対ルール」
  • 1. キャッチコピー・基本情報の変更
  • 2. ニュース（お知らせ）の追加
  • 3. 研究内容（Research）の変更

編集時の３つの「絶対ルール」

  • 文字を囲んでいる " "（ダブルクォーテーション） は絶対に消さないでください。
  • 項目の最後にある ,（カンマ） は絶対に消さないでください。
  • 記号（" や , など）は必ず「半角」で入力してください。全角を使用するとプログラムが停止し、画面が真っ白になります。

1. キャッチコピー・基本情報の変更

トップページを開いたときに表示される文章や、About Us（研究室概要）の文章を変更します。

hero: { catch: "巨大分子を自在に組み上げ、新たな機能と構造を創る", }, about: { title: "About Us", desc: "世の中で実用化されている機能性物質の開発では...\n\n我々の研究グループのモチベーションは...", }
改行のテクニック：
文章の途中で改行したい場合は、Enterキーではなく \n （半角バックスラッシュと小文字のn）を入力します。段落を分けたい場合は \n\n と2回入力してください。

2. ニュース（お知らせ）の追加

新しいお知らせは、items: [ ] の中の一番上の行にコピー＆ペーストして追加します。

news: { title: "News", items: [ // ↓新しいお知らせはここに追加します { date: "2026.05.01", text: "新しい論文が掲載されました。", isNew: true }, { date: "2026.04.01", text: "新四年生が配属されました。", isNew: false }, ] }
  • date : 日付を半角で入力します。
  • text : お知らせの内容を入力します。
  • isNew : true で赤い「New」マークが付きます。不要になれば false に変更します。

3. 研究内容（Research）の変更

研究のテーマタイトルや説明文を変更できます。画像を変更する場合は、src/img/ フォルダに新しい画像を置き、ファイル名を書き換えます。

research: { title: "Research", items: [ { img: "src/img/research_img_01.png", title: "有機合成や錯体合成による巨大なπ拡張系分子の創製", desc: "有機化学や錯体化学の合成法を利用して..." }, // ... ] }

4. メンバー（経歴・学生）の更新

教員の経歴（History）や、所属学生（Students）の人数・名前を更新します。

// 経歴の追加 history: [ { year: "2026.01", event: "岐阜大学 工学部 ... 准教授（現職）" } ], // 学生の追加・変更（行をコピーして増やせます） students: [ { grade: "M1 (修士1年)", count: "1名<br>・岐阜 太郎" }, ←最後のカンマ必須 { grade: "B4 (学部4年)", count: "2名<br>・山野 拓海<br>・松尾 暁和" } ]
リスト内の改行：
学生の名前を列挙するなど、枠の中で改行したい場合は <br> と入力してください。

5. 業績リンク（Publications）の変更

researchmapなどの外部リンクURLを変更します。

pub: { title: "Publications", linkMsg: "こちらをご覧下さい。", url: "https://researchmap.jp/7000016501?lang=ja" }

6. 英語版ページ（EN）の更新について

data.js の後半には en: { ... } というブロックがあります。
日本語版（ja:）でニュースやメンバーを追加した際は、必ずこちらの英語版（en:）の該当箇所にも英語で同様の追加を行ってください。

作業完了後の確認

data.js を上書き保存し、ご自身のパソコンで index.html を開いてみてください。
画面が真っ白になっていなければ成功です！表示内容を確認後、サーバーへアップロードしてください。
